# RapidRoster MVP

## Goal

Build a web application that allows organizers to import participant data, configure group generation rules, and automatically generate optimal groupings.

The MVP should focus on solving one problem well: creating high-quality groupings. Features that do not directly contribute to this goal should be postponed until later versions.

---

# Wizard

Single page wizard-like flow with progress indicator.

```
RapidRoster

    - Configure participants

    - Configure rules

    - Generate groupings

    - Review results

    - Export results
```

---

# Success Criteria

The MVP is complete when an organizer can:

- Import participants from a CSV file.
- Configure group generation rules.
- Generate optimal groupings.
- Review generated groupings.
- Export the final groupings.

If these tasks can be completed smoothly, the MVP has achieved its primary goal.
